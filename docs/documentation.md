
# Estrategia Demonio de Shannon
Se van a dividir los activos en dos tipos, acciones y demonios. Las acciones son el motor principal de crecimiento, idealmente vender lo menos posible. Los demonios sólo se usan para aprovechar su volatilidad.
## Proporciones
Las proporciones entre las acciones y los demonios no va a ser fija, sino que van a fluctuar un poco dependiendo del caso. Como base, se tiene una proporción 80/20 entre acciones y demonios, pero puede variar +-5 dependiendo de ciertas condiciones.
### Acciones
Para las acciones, se va a invertir principalmente en acciones globales, geográficamente neutras, usando el ACWI como proxy, y con fuertes tilts hacia factores de rentabilidad, small cap y value.

### Demonios
Los demonios no tienen ningún valor en si, por lo que no se usarían para inversiones a largo plazo, sólo se utilizan para aprovechar su gran volatilidad y baja o negativa correlación con las acciones.

Principalmente se usarán bonos a largo plazo de EEUU, metales preciosos, criptomonedas y commodities. Las proporciones de cada uno es variable, pero tendrán como mínimo 16% y máximo 25% entre todos los demonios.

## Rebalanceo
Para rebalancear los distintos activos, se calculará una variación del 20% relativo de cada proporción, y se verificará que la variación significará una transacción de más de 50 USD, sólo ahí se activará la alerta de compra / venta. Esto para evitar perder dinero en los spreads de cada activo.

Además, se debe seguir el siguiente orden de prioridad:
1. Vender Demonios para comprar Acciones
2. Vender Demonios para comprar otros Demonios
3. Comprar Acciones con capital extra de tus ahorros
4. Comprar Demonios con capital extra de tus ahorros
5. Vender Acciones para comprar otras Acciones
6. Vender Acciones para comprar Demonios
